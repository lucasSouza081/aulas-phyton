### Operadores aritiméticos
#### Operadores logicos/comparativos
- **Estruturas** **_condicionais_** (if/elif/else)

- **Tipos de estruturas condicionais:**

- if: Executar um bloco de código se ima condição especifica for verdadeira.
- elif: Significa "else if" e permite que você verifique várias condições.
- else: Excuta um bloco de código se nenhuma condições anteriores for verdadeora.

```py
 tereza = 1.80
 marcelo = 1.80

 #Se for positivo, o print será ativado
 if marcelo > tereza:
  print("Marcelo é mais alto")
 elif Marcelo == tereza:
  print("Marcelo é tereza")
 else: 
  print("Tereza é mais alta")
